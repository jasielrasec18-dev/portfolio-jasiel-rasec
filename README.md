# Jasiel Rasec — Portfólio Pessoal

![status](https://img.shields.io/badge/status-active-brightgreen) ![license](https://img.shields.io/badge/license-MIT-blue)

## Descrição
Portfólio pessoal desenvolvido com React + TypeScript e Vite para apresentar projetos, certificações e canais de contato. Foco em UI responsiva, animações e manutenção simples por meio de dados em `src/data/`.

## Demo
- Url Deploy: https://portfolio-jasiel-rasec.vercel.app/

## Tecnologias
- TypeScript
- React
- Vite
- TailwindCSS
- GSAP (animações)
- ESLint

## Scripts úteis
- `npm run dev` — servidor de desenvolvimento  
- `npm run build` — compilação para produção (`tsc -b && vite build`)  
- `npm run preview` — pré-visualizar o build  
- `npm run lint` — executar ESLint

## Instalação e execução local
```bash
npm install
npm run dev
# build + preview
npm run build
npm run preview
